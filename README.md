# 💧 Hidrate.se

> **Sua garrafa de água digital para controle de hidratação.**

O **Hidrate.se** é um web app focado em ajudar usuários a monitorarem seu consumo de água de forma visual e intuitiva. Através de uma interface interativa com uma garrafa de água digital, o sistema calcula quanto falta para atingir a meta personalizada.

## 🚀 Funcionalidades

- [x] **Meta Personalizada:** Defina sua meta em ml.
- [x] **Garrafa Interativa:** Visualização em tempo real do nível da água em uma garrafa de água digital.
- [x] **Cálculo Automático:** Sistema que subtrai o consumo da meta restante.
- [x] **Responsividade:** Design adaptado para dispositivos móveis e desktop.
- [x] **Persistência Local:** Salva os dados da meta e da água ingerida localmente até que o usuário "esvazie" a garrafa.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e Metatags SEO.
- **CSS3:** Estilização moderna e layout responsivo.
- **JavaScript (Vanilla):** Lógica de manipulação do DOM e cálculos de consumo.
- **SVG:** Gráficos vetoriais para a interface da garrafa.

## 💻 Como usar

1. Clique no botão de menu no lado superior direito da tela.
2. Insira sua meta no campo "META".
3. Digite a quantidade de água ingerida no campo "ÁGUA INGERIDA".
4. Clique em **BEBER** para ver a garrafa encher e o contador atualizar.
5. Use o botão **ESVAZIAR** para esvaziar a garrafa e resetar o seu progresso.
