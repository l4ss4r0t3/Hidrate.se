Sua garrafa de água digital. Acompanhe sua hidratação diária de forma visual e interativa — funciona no navegador, pode ser instalado como app e salva seus dados na nuvem.

---

## Como usar

**1. Defina sua meta diária**
O padrão é 2000ml. Você pode alterar no campo "Sua Meta Diária" ou usar a **Calculadora** (Menu → Utilidades → Calculadora) para estimar com base no seu peso.

**2. Registre cada vez que beber**
Digite a quantidade em ml no campo "Água a Beber" e toque em **Beber Água**. A garrafa vai esvaziando conforme você se hidrata.

**3. Acompanhe o progresso**
A garrafa mostra quanto falta para bater a meta. Ao completar, aparece uma mensagem de celebração 🎉

**4. Novo dia**
O app reseta automaticamente à meia-noite e salva o dia anterior no histórico.

## Funcionalidades

| Funcionalidade | Descrição |
|---|---|
| 💧 Garrafa interativa | Visual que esvazia conforme você bebe |
| 🎯 Meta personalizável | Defina ou calcule sua meta diária |
| 📚 Histórico | Últimos 7 dias com registros individuais |
| ☁️ Sincronização | Login com Google salva tudo na nuvem |
| 📴 Offline | Funciona sem internet, sincroniza ao voltar |
| 🎨 Temas | Modo escuro, papel de parede e 3 modelos de garrafa |
| 📱 Instalável | Funciona como app no Android e iOS |

---

## Login com Google

Sem login, seus dados ficam salvos apenas no dispositivo atual.

Com login:
- Dados sincronizados entre dispositivos
- Histórico preservado na nuvem
- Migração automática dos dados locais ao logar pela primeira vez

---

## Instalar como app

**Android:** Abra o site no Chrome → menu (⋮) → "Adicionar à tela inicial"

**iOS:** Abra no Safari → botão compartilhar → "Adicionar à Tela de Início"
